RustyNES **v1.0.0** is the first stable release: a **cycle-accurate** Nintendo Entertainment System emulator in pure Rust, with a polished desktop shell and a live in-browser demo.

**[Try it in your browser](https://doublegate.github.io/RustyNES/)** — no install required.

## Highlights

This release replaces RustyNES's emulation core with a new **master-clock-precise, lockstep-scheduled engine** and ports the desktop UX onto it.

- **Reference-grade accuracy** — **AccuracyCoin 100.00% (139/139)**, `nestest` 0-diff vs the Nintendulator golden log, region-exact 3:1 NTSC/Dendy and 3.2:1 PAL timing. A strict bit-determinism contract underpins save-states, the regression oracles, TAS movies, and rollback netplay.
- **51 mapper families** — NROM through MMC5, the full VRC line (incl. VRC6 + VRC7 OPLL FM audio), Sunsoft FME-7 (+ 5B), Namco 163, and the Taito/Irem/Jaleco/Bandai/Tengen/Konami long tail.
- **Famicom Disk System** — real-BIOS boot, read/write drive, writable `.fds.sav`, 2C33 wavetable audio.
- **Vs. System / PlayChoice-10** — true 2C03/2C04/2C05 RGB arcade games with per-game DIP presets.
- **Rollback netplay** — GGPO-style, 2–4 players, native UDP + a browser WebRTC mesh.
- **RetroAchievements** (opt-in, native) — achievements, leaderboards, rich presence, hardcore mode.
- **TAS movies**, **save states + rewind**, **run-ahead** latency reduction, **display-sync pacing**, a **lock-free audio ring** with dynamic rate control, and a **read-only egui debugger** (CPU/PPU/APU/memory/OAM/mapper).
- **WebAssembly build** running the exact same core in the browser (GitHub Pages demo above).

## Desktop UX

A complete egui shell: an always-on menu bar + status bar, a recent-ROMs list, a tabbed Settings window (Display/Audio/Input/Advanced), light/dark/system themes, 8:7 pixel-aspect and optional overscan cropping, integer window-size presets, a first-run Welcome modal, and drag-and-drop ROM loading. Plus master volume + per-APU-channel mutes, **25 %–300 % emulation-speed presets** (with pitch-shifted audio), hold-to-fast-forward, single-frame advance, a **thumbnail save-state manager**, screenshot-to-file/clipboard, a gamepad deadzone control with hot-plug detection, and a pause-dim overlay.

Default keys: arrows + Z/X/Enter/R-Shift (P1), Space pause, F1/F4 save/load, F5 rewind, Tab fast-forward, `\` frame-advance, `=`/`-`/`0` speed, F11 fullscreen, F12 open, `` ` `` debugger. All TOML-rebindable.

## Install

Pre-built binaries for Apple-Silicon (`aarch64`) macOS, `x86_64` Linux, and `x86_64` Windows are attached below. Or build from source (Rust 1.86): `cargo run --release -p rustynes-frontend -- path/to/rom.nes`. Linux needs `libxkbcommon`, `wayland`, `alsa-lib`, and `systemd-libs` (udev). See the [README](https://github.com/doublegate/RustyNES#readme) for per-platform details and controls.
