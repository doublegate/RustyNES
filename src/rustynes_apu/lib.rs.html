1//! `RustyNES` APU - Nintendo Entertainment System Audio Processing Unit Emulation
2//!
3//! This crate provides a cycle-accurate emulation of the NES 2A03 APU (Audio Processing Unit).
4//!
5//! # Features
6//!
7//! - **5 Audio Channels**:
8//!   - 2× Pulse channels (square waves with sweep)
9//!   - 1× Triangle channel (triangle wave)
10//!   - 1× Noise channel (pseudo-random noise)
11//!   - 1× DMC channel (Delta Modulation Channel for sample playback)
12//!
13//! - **Hardware-Accurate Components**:
14//!   - Frame counter with 4-step and 5-step modes
15//!   - Envelope generators
16//!   - Length counters
17//!   - Sweep units
18//!   - Non-linear mixing
19//!
20//! - **Zero Unsafe Code**: No `unsafe` blocks (enforced by `#![forbid(unsafe_code)]`)
21//!
22//! # Example Usage
23//!
24//! ```rust
25//! use rustynes_apu::Apu;
26//!
27//! let mut apu = Apu::new();
28//!
29//! // Enable pulse 1
30//! apu.write_register(0x4015, 0x01);
31//!
32//! // Configure pulse 1: 50% duty, constant volume 15
33//! apu.write_register(0x4000, 0xBF);
34//!
35//! // Set frequency (A4 = 440 Hz)
36//! // Timer = CPU_CLOCK / (16 * frequency) - 1
37//! // Timer = 1789773 / (16 * 440) - 1 = 253
38//! let timer: u16 = 253;
39//! apu.write_register(0x4002, (timer & 0xFF) as u8);
40//! apu.write_register(0x4003, ((timer >> 8) & 0x07) as u8);
41//!
42//! // Step the APU each CPU cycle
43//! for _ in 0..1000 {
44//!     apu.step();
45//! }
46//! ```
47//!
48//! # Architecture
49//!
50//! The APU consists of several interconnected components:
51//!
52//! - **Frame Counter** (`frame_counter`): Times envelope, length counter, and sweep updates
53//! - **Envelope Generator** (`envelope`): Controls volume fade-in/fade-out
54//! - **Length Counter** (`length_counter`): Automatically silences channels after a duration
55//! - **Sweep Unit** (`sweep`): Modulates pulse channel frequencies
56//! - **Channels** (`pulse`, `triangle`, `noise`, `dmc`): Individual audio generators
57//! - **Mixer** (`mixer`): Combines channel outputs with non-linear mixing
58//! - **Resampler** (`resampler`): Converts APU rate to target audio sample rate
59//!
60//! # Register Map
61//!
62//! The APU is controlled via memory-mapped registers at CPU addresses `$4000-$4017`:
63//!
64//! | Address | Channel | Register |
65//! |---------|---------|----------|
66//! | `$4000-$4003` | Pulse 1 | Duty, sweep, timer, length |
67//! | `$4004-$4007` | Pulse 2 | Duty, sweep, timer, length |
68//! | `$4008-$400B` | Triangle | Control, timer, length |
69//! | `$400C-$400F` | Noise | Envelope, period, length |
70//! | `$4010-$4013` | DMC | Flags, load, address, length |
71//! | `$4015` | Status | Enable/status |
72//! | `$4017` | Frame Counter | Mode, IRQ inhibit |
