# RustyNES Desktop

Desktop frontend for RustyNES NES emulator, built with Iced framework.

## Features

- Iced 0.13+ UI framework with Elm architecture
- wgpu-powered GPU rendering
- Native file dialogs for ROM loading
- Cross-platform audio via cpal
- Gamepad support with gilrs

## Usage

```bash
cargo run -p rustynes-desktop
```
