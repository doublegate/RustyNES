# Milestone 15: Video Filters

**Phase:** 4 (Polish & Release)
**Duration:** Months 20-21
**Status:** Planned
**Target:** August 2027

---

## Overview

Implement authentic video filters for enhanced visual experience. Includes NTSC filter, CRT shaders, palette options, and aspect ratio modes.

## Goals

- [ ] NTSC filter (Blargg)
- [ ] CRT shader (scanlines, curvature, bloom)
- [ ] Palette options (Composite, RGB, Custom)
- [ ] Aspect ratio modes (4:3, Pixel Perfect, Stretch)
- [ ] Overscan cropping

## Acceptance Criteria

- [ ] Filters look authentic
- [ ] <2ms overhead per frame
- [ ] User-adjustable parameters

## Dependencies

- wgpu rendering pipeline mature
- Performance headroom available
