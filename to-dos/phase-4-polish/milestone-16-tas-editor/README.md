# Milestone 16: TAS Editor

**Phase:** 4 (Polish & Release)
**Duration:** Months 17-18
**Status:** Planned
**Target:** June 2027

---

## Overview

Build comprehensive TAS (Tool-Assisted Speedrun) editor with greenzone, bookmarks, piano roll, and branching system. Competitive with FCEUX TAS editor.

## Goals

- [ ] Greenzone (verified frame history)
- [ ] Bookmarks
- [ ] Piano roll input editor
- [ ] Branch system
- [ ] Undo/redo
- [ ] Input recording shortcuts

## Acceptance Criteria

- [ ] Can create/edit TAS movies
- [ ] Greenzone manages 10,000+ frames
- [ ] Branching works reliably
- [ ] Competitive with FCEUX TAS editor

## Dependencies

- TAS recording/playback functional
- Deterministic emulation verified
- GUI extensible for editor panels
