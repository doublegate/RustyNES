# Milestone 9: Lua Scripting

**Phase:** 2 (Advanced Features)
**Duration:** Months 9-10
**Status:** Planned
**Target:** October 2026

---

## Overview

Integrate mlua 5.4 for runtime scripting. This milestone enables users to create custom tools, bots, visualizations, and gameplay modifications through Lua scripts.

## Goals

- [ ] mlua 5.4 integration
- [ ] Memory read/write API
- [ ] Callback hooks (frame, scanline, instruction)
- [ ] Input injection
- [ ] GUI overlay support
- [ ] Example scripts (hitbox viewer, bot AI)

## Acceptance Criteria

- [ ] Can read/write RAM from Lua
- [ ] Frame callbacks work at 60 Hz
- [ ] Drawing primitives render correctly
- [ ] <5% performance overhead

## Dependencies

- Core emulation stable
- GUI rendering pipeline extensible
