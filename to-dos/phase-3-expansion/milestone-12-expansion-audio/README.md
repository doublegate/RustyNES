# Milestone 12: Expansion Audio

**Phase:** 3 (Expansion)
**Duration:** Months 13-15
**Status:** Planned
**Target:** March 2027

---

## Overview

Implement expansion audio chips for enhanced sound capabilities. Supports VRC6, VRC7, MMC5, Namco 163, Sunsoft 5B, and FDS audio.

## Goals

- [ ] VRC6 (2 pulse + sawtooth)
- [ ] VRC7 (FM synthesis)
- [ ] MMC5 (2 pulse + PCM)
- [ ] Namco 163 (8 wavetable channels)
- [ ] Sunsoft 5B (3 square + noise)
- [ ] FDS (wavetable + modulation)

## Test Games

- Castlevania III (VRC6)
- Lagrange Point (VRC7)
- Castlevania (FDS)

## Acceptance Criteria

- [ ] Expansion audio sounds accurate
- [ ] Music matches hardware recordings
- [ ] Proper channel mixing

## Dependencies

- APU core complete
- Mapper framework extensible for audio
