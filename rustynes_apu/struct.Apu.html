Expand description

Main APU (Audio Processing Unit) structure

Integrates all 5 audio channels (2 pulse, triangle, noise, DMC) and the frame counter. Provides memory-mapped register interface at CPU addresses $4000-$4017.

pub fn step(&mut self) -> u8

Steps the APU by one CPU cycle

This should be called once per CPU cycle to keep the APU synchronized. Generates audio samples internally which can be retrieved via samples().

§Returns

Number of DMC DMA stall cycles (0 if no DMA occurred, typically 3 when DMA happens)

§Panics

Panics if memory_read_fn is not set when DMC attempts DMA

Source

pub fn step_with_action(&mut self) -> FrameAction

Steps the APU by one CPU cycle and returns frame action

This is a convenience method that returns the frame action instead of DMA stall cycles. Use step() when you need to handle DMC DMA stalls.

§Returns

Frame action to be processed (if any)
